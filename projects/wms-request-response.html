Notes

  • The dasboard displays the status of the latest 500 items in the ingestion-catalog workflow.
  • To download the XML of a feed, click the link "Download Feed". You will be taken to an error screen if there is something wrong with the download
  • To reprocess a feed, click the link "Reprocess Feed". A screen will pop either confirming that the feed is being reprocessed, or displaying an error if the screen could not be reprocessed.
  • You can filter the displayed results by typing into the box labeled "Filter" Below. Filtering happens interactively. Filtering is available for DMIDs, ASINs, External Ids, and any of the process states. For example, to see all the items that have failed status, type "failed" into the filter box
